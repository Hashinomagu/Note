- プログラムの書き方
  - 下記のように分ける
  - 仕様書
    - インターフェース
  - 実装
    - クラスとして記載
- 図形の仕様書
  - 取り扱う図形
    - 三角形、円、多角形
  - メソッド
    - 図形の描画
    - 図形の移動
    - 図形の説明文を返す

  '''
    public interface Figure{
      //図形の描画
      void draw();

      //図形の移動
      void move(double dx, doble dy);

      //図形の説明文を返す
      String toString();
    }
  '''
  - class　ではなく interface
- インタフェースの利用
  - 仕様に従っていることを明記してクラスを定義できる
    ```
      public class クラス名 implements インタフェース名{
        以下はいつも通り
      }
    ```
