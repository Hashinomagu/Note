- ### for文
    ```
        for(初期化;条件式;更新){
            
        }
    ```
- While文
  - 特徴
    - for文の条件式のみ持ってきてたも
    - 先に条件識別を行う
      - 条件に合わない場合は実行されないこともある
      - 実行させるための条件
    ```
        while(条件式){

        }
    ```
- do while文
  - 特徴
    - 必ず1回は実行される
    - 最後に条件識別を行う
      - いわば終了条件の定義みたいなもの
    ```
        do{
            
        } while(条件式);
    ```
