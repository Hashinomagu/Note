- ## オブジェクト
  - ひとまとめにするもの
  - オブジェクトを使うには**設計図**が必要
    - 分子分母をまとめて記憶
    - 新しい分数オブジェクtを作る
    - 表示する
    - ２つのオブジェクトの足し算ができる
  - 例
    - 下記のプログラムを例として進めます
        ```
                //mainのクラス（実行するクラス）
            class A{
                public static void main(String[] args){
                    Bunsu b1=new Bunsuu(5,3); 
                    Bunsu b2=new Bunsuu(1,2);
                    b1.println();


                }
            }


            // Bunsuuクラス（設計図のクラス)
            public class Bunsuu{
                int k;
                int n;
                Bunsu(int kk, int nn){
                    k=kk;
                    n=nn;
                }
                void println(){
                    System.out.println(k+"/"+n);
                }
            }
        ```  
    - 1行目の処理
        ``` 
            Bunsu b1=new Bunsuu(5,3); 

            [これが呼び出される]
            Bunsu(int kk, int nn){
                k=kk;
                n=nn;
            }
        ```  
        ```
            [イメージ]
                Bunsu b1=new Bunsuu(5,3); 
                    kk<5
                    nn<3

                    ------k=5 <kk(5)
                    |   --n=3 <nn(3)
                    |   |
                    |   |
                  | k5  n3   |
                b1|__________|
        ```
    - 2行目の処理
        ``` 
            Bunsu b2=new Bunsuu(3,5); 

            [これが呼び出される]
            Bunsu(int kk, int nn){
                k=kk;
                n=nn;
            }
        ```  
        ```
            [イメージ]
                                    Bunsu b2=new Bunsuu(3,5); 
                                        kk<3
                                        nn<5

                                        ------k=3 <kk(3)
                                        |   --n=5 <nn(5)
                                        |   |
                                        |   |
                   | k5  n3   |.     | k3  n5   |
                 b1|__________|    b2|__________|
        ```
    - 3行目の処理
        ``` 
                b1.println();

            [これが呼び出される]
                void println(){
                    System.out.println(k+"/"+n);
                }
        ```  
        ```
            [イメージ]

                   | k5  n3   |.     | k3  n5   |
                 b1|__________|    b2|__________|
                      / \
                     /   \
                    /_____\  
                      | |   
                      | |   
                      | |   
                    こいつを表示

            [たぁ〜みなる]
            5/3
        ```
- クラスの定義の構文
    ```
        public class クラス名{
            //インスタンス変数の宣言
            int a;
            string b;



            //コンストラクタの定義
            /*

            */
            クラス名(仮引数){

            }


            //メソッド
            /*
                staticを書かない
            */
            void A(){

            }
            int B(int b1){

            }
        }
    ```
- 呼び出し等
  - b1のAメソッドを実行したい
    - b1.A();
  - b2のインスタンス変数kを見たい
    - int a= b2.k;
  - 上記のように「の」を示すには「.」 を使う
