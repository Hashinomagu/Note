## 配列
#### 宣言
    ```
        例int 型の場合

        int[] 変数名 =new int[要素数]
    ```
#### 宣言＋初期値の登録
    ```
        int [] 変数名= {1,2,3};
    ```
#### 代入
    ```
        変数名[要素番号]=1;
    ```

#### 参照
    通常の変数への代入
        ```
            a= 変数名[要素番号];
        ```
    配列への代入
        ```
            int[] a=new int[5];
            配列a=配列b;
        ```
