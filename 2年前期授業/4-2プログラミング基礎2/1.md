## 演算
- 整数＋整数　＝＞整数
- 実数/実数　=>実数
- 実数/整数 =>実数/実数=>実数
- 整数+文字列　->文字列＋文字列>文字列
- ワイドニング変換
  - 範囲が大きい方に変換
  - 何も書かなくてもいい
- ナローイング変換
  - 範囲が小さい方に変換
  - (int)などといったキャスト構文を書く必要がある
  - データが変更される必要がある

## 分岐構造
- ### if文
    ```
    int y=2026;
    if(y % 400 == 0  ||  y % 100!=0 && y % 4 == 0){
        真の時
    }else
    {
        偽の時
    }
    ```
-  ### switch文
   -  分岐構造の一つ
      -  if文:2分岐（then/else）
      -  switch文:n分岐
        ```
          switch(式){
            case 値:
              文
            case 値:
            case 値:
              文
            default:
              文
          }
        ```
      - switchで式を定義
      - caseで条件を定義
      - defaultがそのほか
      - breakで外に飛ばす
      - 指定可能な型
        - char,byte,short,int,String
      - 文には0以上の命令が書ける 

  - 例1 
    ```
    int a=1;
    switch (a){
        case 1:
          System.out.println("A");
        case 2:
          System.out.println("B");
        case 3:
          System.out.println("C");
        default:
          System.out.println("D");

    }
    ーーーーーーーーーーーーーーーーーーーーーー
    出力結果
    A
    B
    C
    D
    ```
  - 例2
    ```
    int a=9;
    switch (a){
        case 1:

          System.out.println("B");

    ーーーーーーーーーーーーーーーーーーーーーー
    ```

    ```
          System.out.println("D");

    ーーーーーーーーーーーーーーーーーーーーーー
    出力結果
    A
    ```
