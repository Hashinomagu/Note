# アクセス制御と不変オブジェクト
- ## アクセス制御とは
  - オブジェクトの外からインスタンス変数とメソッドに**アクセスできる**か**アクセスできない**かを制御する
  - アクセスできないのにアクセスしようとしたらコンパイルエラーになる
  - #### public
    - 外からアクセス可能
    - 例
      - public static void main(string[] args)
      - purblic int a;
  - #### private 
    - 外からアクセスできない
    - 例
      - private int b;
  - 何も書かない
    - 条件付きで他のクラスに公開
    - 条件とは？
      - 同じパッケージのクラス
        - パッケージとは？
          - javaではパッケージ構文がある同じ用途のプログラムを一つのフォルダにまとめて管理する
          - 同じフォルダの中のものをまとめて管理する
  - protected
    - 条件付きで他のクラスに公開
- ## アクセスの構文
  - クラス名・メソッド名・変数名の前につけてあげる
- ## 演習1
    ```
        次のプログラムの中でコンパイルエラーが発生するクラス名と行番号を解答しましょう。
        【A.	java】
        1: public class A {
        2:     public int m;
        3:     private int n;
        4:     public A(int mm, int nn) {
        5:         m = mm;
        6:         n = nn;
        7:     }
        8:     private int getM() {
        9:         return m;
        10:     }
        11:     public int getN() {
        12:         return n;
        13:     }
        14: }

        【MainA.java】
        1: public class MainA {
        2:   public static void main(String[] args) {
        3:     A a = new A(3,5);
        4:     System.out.println(a.m);
        5:     System.out.println(a.n);
        6:     System.out.println(a.getM());
        7:     System.out.println(a.getN());
        8:   }
        9: }

       答え:MainA-5・MainA-6
    ```

- 演習におけるポイント
  - 中側ではなく呼び出した側でエラーが出る
  - classの修飾子は**public**または**何も書かない**のみ
  - アクセス制御の一般的な設定
    - インスタンス変数はprivate
      - ただし**継承**されるクラスのインスタンス変数はprotected
    - メソッドはpublic
      - getやSetメソッドを作る
        ```
            public class E{
                private int n;
                public E(int nn){
                    n=nn;
                }
                public int getN(){
                    return n;
                }
                puvlic void setN(int nn){
                    n= nn;
                }
            }
        ``` 
        よくある例 参照のみ可
        ```
            public class E{
                private int n;
                public E(int nn){
                    n=nn;
                }
                public int getN(){
                    return n;
                }

            }
        ```

- 状態が変更できるオブジェクト　可変オブジェクト
- 状態が変更できないオブジェクト　不変オブジェクト
