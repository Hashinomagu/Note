## ターミナルの使い方
```
    makiyuuto@makiyuutonoMacBook-Air ~ %      
```
- ~の意味は？
  - ホームディレクトリの位置
    - 例カーナビで「自宅」を設定しているみたいなmの
- 現在のディレクトリの位置
  - ＞pwdコマンドを使う
    ```
        makiyuuto@makiyuutonoMacBook-Air ~ % pwd  
        /Users/makiyuuto
    ```
- ディレクトリの位置の見方
  - /Users/makiyuutoとあった時
    - |-「Users」
      - |-「makiyuuto」＜ここを参照していることになる
        - |-「ドキュメント」
        - |-「デスクトップ」
      - |-「Hashimgau」
        - |-「ドキュメント」
        - |-「デスクトップ」
    - |-「App」
    - |-「System」
    - |-「library」
- 現在のディレクトリのファイル・フォルダを見る方法
  - lsコマンドを使う
    ```
        makiyuuto@makiyuutonoMacBook-Air Note % ls
        2年前期授業				【個】Csメモ
        GitHubTreatment				【個】HTML系統メモ
        README 1.md				【個】piza復習
        README.md				【個】プログラミング理論
        学年ガイダンス				無題のファイル.md
    ```
    上記の場合
    ```
        Note ＜今見ているところ
            |-2年前期授業	
            |-GitHubTreatment
            |-【個】Csメモ
            |-【個】HTML系統メモ
            |-【個】piza復習
            |-【個】プログラミング理論
            |-学年ガイダンス
            |-README 1.md	
            |-README.md
            |-無題のファイル.md
    ```
    が入っていることを示す
- フォルダの移動
  - cdコマンドを使う
    ```
        makiyuuto@makiyuutonoMacBook-Air ~ % cd Desktop 
        makiyuuto@makiyuutonoMacBook-Air Desktop %  
    ```
    上記のコマンドはホームディレクトリからデスクトップに移動したということを示す
- フォルダの作成
  ```
    makiyuuto@makiyuutonoMacBook-Air Desktop % mkdir データサイエンス   
  ```
- ファイルの作成
  - touchコマンドを使う
    ```
        makiyuuto@makiyuutonoMacBook-Air Desktop % touch test.txt
    ```

  - echoを使う
    ```
        makiyuuto@makiyuutonoMacBook-Air Desktop % echo Hello
    ```

  - catコマンドを使う
    ```
      makiyuuto@makiyuutonoMacBook-Air Desktop % cat test.txt 
      Hello
    ```
