# ネットワーク層
 ## IP層(Internet Protocol)
- パケット交換ネットワーク
  - メッセージをパケットに分割
    - 最後のパケットには宛先のアドレスが書かれてる
    - 分割されたパケットは個別にネットワーク上を移動
- パケットの中継とプロトコル階層
  - TCP
    - 通信の両端のホストで動く
    - IPを使って通信を行う
  - IP
    - 全ての中継ノードで動く
    - 無線LANのような計算機と計算機を中継するようなプロトコルを使う
    - 1対1を繰り返す
- カプセル化
  - 上位層のパケット（ヘッダ+データ）
  - 下位層のパケット　データとして扱う
  - TCP　セグメント
  - IP データグラムという
- IPが提供するサービス
