## 文字列を出力
入力値

    Console.WriteLine("hello world");
出力結果

    hello world

## インデントとは
    using System;
    public class Hello{
        public static void Main(){
            Console.WriteLine("こんにちは");
        }
    }

対応する箇所で字下げをすること
タブキー故意に字下げをしたい場合はスペースキー
またはタブキーを使うことで字下げができる

字下げを行うことでコードが見やすくなる



## 間違いやすいポイント
- 末尾の;忘れ
- ()や{}を全角で入力する
- 大文字、小文字の間違い
