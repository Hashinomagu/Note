#　ユーザーコントロールをリストのように扱う
[C#個人用メモ](0Csメモ.md)
## 関連項目
## コード（全文）
    List<ItemControl> items = new List<ItemControl>();

    private void AddItem()
    {
        var item = new ItemControl();

        item.Location = new Point(10, items.Count * 50);

        this.Controls.Add(item);

        items.Add(item);
    }


## コードについての説明
- #### 1Listの定義 
  ```
    List<ItemControl> items = new List<ItemControl>();
  ```
  - 作成した ItemControl を全部記録しておく箱
  - 今何個あるか数えることができる
  - 後で削除、更新ができる

- #### 2 コントロールの生成
    ```
    var item = new ItemControl();
    ```
  - 新しいUI部品を作る
- #### 3 位置を決める
  ```
  item.Location = new Point(10, items.Count * 50);
  ```
  - X座標→左から 10px
  - Y座標→上から コントロールの数 × 50px

- #### 4 UIに追加
  ```
  this.Controls.Add(item);
  ```
    - フォームに追加する
- #### 5リストに追加
  ```
   items.Add(item);
  ```
  - リストに記録する
