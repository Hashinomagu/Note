## JSONデータの取り扱い
- ### 共通して使用するサービス
    ```
        using System.Text.Json;
    ```
- ### JSON>クラス
  ```
    var json = File.ReadAllText(@"C:\hoge.json");
    var hoge = JsonSerializer.Deserialize<Hoge>(json);

  ``` 
- ### クラス>JSON
    ```
        var hoge = new Hoge();
        hoge.id = 1;
        hoge.name = "hoge";
        var json = JsonSerializer.Serialize<Hoge>(hoge);
        File.WriteAllText(@"C:\hoge.json", json);
    ```
