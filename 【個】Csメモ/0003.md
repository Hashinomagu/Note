## 内容
- 共通定義
    当ノートは各操作において次のものが定義されているものとします
    ```
    using System;
    using System.IO;
    ```
    ```
    string path="ファイルパス";
    ```
    - 共通定義の説明 
      - using System;
            コントロールパネルや基本設定などのような場所。
            インターネットの利用やキー操作の仕様などといったコンピュータとして動かすためのツール一式が入っている 
            Systemに定義されている主要な項目
            - int
            - string
            - bool
            - Console.WriteLine
            - Math
            - DateTime.Nowなど
      - using System.IO;
    
- 各種コード
  - 上書
      ```
      File.WriteAllText(path, "出力情報");
      ```
  - 追記
      ```
      File.AppendAllText(path,"出力情報");
      ```
