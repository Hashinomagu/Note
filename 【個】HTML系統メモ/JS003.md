## JSONからデータを持ってくる方法


```
    {
    "name": "Taro",
    "age": 25,
    "skills": ["JavaScript", "HTML", "CSS"]
    }    
```
```
    fetch("./data.json")
    .then(response => response.json())
    .then(data => {
        console.log(data);

        console.log(data.name);   // Taro
        console.log(data.age);    // 25
        console.log(data.skills); // ["JavaScript", "HTML", "CSS"]
    })
    .catch(error => {
        console.error("JSONの読み込みに失敗しました:", error);
    });
  ```
