## ロード後の処理
書き方
```
    document.addEventListener("DOMContentLoaded", function () {
        // ページが開いたら行いたい処理
    });
```


例
```
    <script>
    document.addEventListener("DOMContentLoaded", function () {
        // ページが開いたら行いたい処理
        console.log("ページが読み込まれました");

        document.getElementById("message").textContent = "表示されました";
    });
    </script>
```
