

#### コードの書き方
```
    <Script>
        const 変数名 = document.getElmentById("HTMLで定義したID");

        変数名.textContent="変更文字列";
    </Script>
```
#### 実行例
ボタンを押したらラベル・ボタンのテキストを書き換える
```
    <label id="statusLabel">未処理です</label>
    <button id="changeButton">変更する</button>

    <script>
    const label = document.getElementById("statusLabel");
    const button = document.getElementById("changeButton");

    button.addEventListener("click", function () {
        label.textContent = "処理済みです";
        button.textContent = "変更しました";
    });
    </script>
```
